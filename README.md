# Backlog Tracker

Inventory, supplier, customer, and order tracking system.

## Tech Stack
- Java
- Spring MVC
- Hibernate / JPA
- JSP + jQuery + MDB
- PostgreSQL
- Wildfly


## Features
- Inventory management
- Purchase & customer orders
- Stock movement tracking
- Role-based access

## Setup
1. Configure database [backlogtracker.sql]
2. Update standalone.xml for jdbc connection and email
3. Run on Wildfly
